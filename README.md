#  smolDiffusion

A mini Latent Diffusion model built for edge hardware because not all of us have massive server farms. 

I trained this using LoRA on a carefully chosen subset of the **LAPIS** dataset. It runs surprisingly well and can generate an image in about 3 minutes on a normal GPU.

### Try it out
 **[smolDiffusion on Hugging Face Spaces](https://huggingface.co/spaces/AnshT0612/smolDiffusion)**
